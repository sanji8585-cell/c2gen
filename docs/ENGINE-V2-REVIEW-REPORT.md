## CRITICAL (반드시 수정)

### 1. 괄호 파서가 일반 괄호를 디렉티브로 오인 [QA+아키텍트]
- `GDP(국내총생산)` 같은 일반 괄호를 디렉티브로 파싱함
- 수정: `키:` 패턴이 있거나 DIRECTIVE_KEY_MAP에 등록된 플래그일 때만 디렉티브로 인식
- 중첩 괄호 `(배경: (매우 어두운) 사무실)` 미지원 명시 필요
- 빈 괄호 `()`, 값 없는 키 `(화자:)` 등 엣지케이스 처리

### 2. 디렉티브 텍스트가 TTS에서 그대로 읽힘 [QA]
- 현재 plan에 "TTS 호출 전 디렉티브 strip" 단계가 누락
- ElevenLabs가 괄호를 자동 스킵한다는 가정은 불확실
- 수정: `parseDirectives()` → `cleanNarration` 추출 → TTS에 cleanNarration만 전달

### 3. handleGenerateAdvanced() 380줄 코드 중복 [아키텍트]
- 기존 handleGenerate() 복사 = 유지보수 악몽
- 수정: `services/generationOrchestrator.ts` 공유 서비스 레이어 추출
  - runAudio(), runImages(), runAutoBgm() 등 재사용 가능한 함수로 분리
  - handleGenerate()와 handleGenerateAdvanced()가 같은 오케스트레이터 호출
  - 차이점(디렉티브 파싱, Voice ID 매핑)만 컨텍스트로 주입

### 4. ElevenLabs 다중 음성 비용 "동일" 주장 오류 [비즈니스]
- 프리미엄 Voice ID는 standard보다 2~3배 비쌀 수 있음
- 실제 ElevenLabs pricing tier 감사 필요
- 수정: 다중 음성 크레딧을 +5크레딧/씬으로 조정하거나 표준 Voice만 기본 제공

### 5. 글로벌 assetsRef 공유로 인한 모드 간 암묵적 커플링 [아키텍트]
- assetsRef가 전역이라 고급 모드 ↔ 기존 모드 간 상태 오염 가능
- 수정: 별도 advancedAssetsRef 또는 React Context 분리

## HIGH (출시 전 해결)

### 6. AI 어시스턴트를 메인 진입점으로 전환 [UX+비즈니스]
- 현재: 디렉티브 문법이 메인 → AI 어시스턴트가 보조
- 수정: AI 어시스턴트가 기본 진입점, 직접 디렉티브 작성은 "파워유저 모드"로 토글
- 사용자는 의도만 쓰고 AI가 대본 완성 → 편집 가능 → 생성

### 7. 순차 렌더링 타임아웃/취소 전략 없음 [QA+아키텍트]
- 씬1 실패 → 씬2~N 무한 대기 (데드락)
- 수정: 씬별 120초 타임아웃, 실패 시 독립 모드 폴백, 취소 버튼

### 8. 잘못된 Voice ID → 무음 폴백 (사용자 모름) [QA]
- 삭제/잘못된 Voice ID → Gemini TTS 폴백 → 다른 성별/언어 음성
- 수정: Voice ID 유효성 검증 + 사용자 알림 + Gemini 폴백도 언어/성별 매핑

### 9. 커뮤니케이션 계획 완전 부재 [PM]
- 베타/출시/사용자 안내 전략 없음
- 수정: Soft Launch → Public Beta → GA 3단계 계획 추가

### 10. 테스트 전략 없음 [아키텍트+PM]
- "검증" = 수동 확인만
- 수정: directiveParser 단위 테스트 20+, 통합 테스트, 회귀 테스트

### 11. 프로젝트 마이그레이션 경로 없음 [PM+QA]
- V9.2 프로젝트에 directives 필드 없음 → V2.0에서 로드 시 에러 가능
- 수정: 스키마 버전 필드 + 마이그레이션 함수 + 다운그레이드 보호

### 12. MVP 범위 미정의 [PM+비즈니스]
- 6개 Sprint 전부 = V2.0? 아니면 어디까지가 MVP?
- 추천 MVP: Sprint 0 + 1 + 1.5 (탭 + 디렉티브 + AI 어시스턴트)

## MEDIUM (개선 권장)

### 13. 자막 색상 = 색상만 의존 (접근성) [UX]
- 색맹 사용자 대응: 색상 + 라벨 `[남자] 텍스트` 병행
- WCAG 명도 대비 4.5:1 이상 보장

### 14. 순차 모드 20+ 씬 메모리 이슈 [QA]
- 이미지 20장 × 3MB = 60MB+ 브라우저 메모리
- 수정: 15씬 이상 경고, 참조 이미지 압축, 사용 후 정리

### 15. 다국어 UI 라벨 미정의 [PM]
- 탭 이름, 설정 라벨 등 3개 언어 버전 필요
- config.ts에 ADVANCED_MODE_LABELS 추가

### 16. Sprint 순서 재검토 [비즈니스]
- 비즈니스 관점: 다중 음성(가장 눈에 띄는 기능)을 Sprint 0 직후로?
- 현재 순서도 기술적으로 합리적이므로 선택의 문제

### 17. SRT 화자 표시 포맷 호환성 [QA]
- `[남자] 텍스트` 형태가 일부 플레이어에서 비호환
- 수정: 옵션으로 화자 표시 ON/OFF

### 18. 모드 전환 중 생성 시 상태 불일치 [QA]
- 고급 모드에서 생성 중 → 수동 모드로 탭 전환 → 상태 오염
- 수정: 생성 중 탭 전환 잠금 또는 확인 대화상자

### 19. 분석/추적 계획 없음 [PM]
- 어떤 디렉티브가 많이 쓰이는지, 채택률 등 측정 불가
- 수정: 이벤트 추적 (advanced_tab_viewed, directive_used 등)

### 20. Playground 공유 시 고급 모드 데이터 처리 [QA]
- 공유된 프로젝트에서 디렉티브/화자 정보 표시 여부
- 수정: 공유 시 메타데이터 포함, 뷰어에서 화자 정보 표시
